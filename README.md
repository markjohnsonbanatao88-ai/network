## WhatsApp MFR conversion

Parsed 5,685 messages from `_chat.txt`, covering February 23, 2026 – July 5, 2026.
Media attachments are rendered as compact placeholders by filename; the large media files from the WhatsApp ZIP are not embedded in this Vercel package.

## Deploy

```bash
vercel --prod
```

Private chats should not be committed to a public GitHub repo unless you are okay with public access.
